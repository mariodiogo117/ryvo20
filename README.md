# Website oficial RYVO

Website institucional de `ryvo.pt`, reformulado para apresentar a RYVO como uma plataforma multidesporto para encontrar pessoas, criar jogos e participar em eventos.

## Estrutura

- `index.html` — página institucional principal
- `css/styles.css` — design responsivo mobile-first
- `js/main.js` — menu móvel e pequenos comportamentos de interface
- `assets/multisport/` — imagens otimizadas da nova campanha multidesporto
- `politica-privacidade.html` — Política de Privacidade
- `termos-condicoes.html` — Termos e Condições

## Abrir localmente

```powershell
py -m http.server 8080
```

Depois abre `http://localhost:8080`.

## Publicação

O projeto continua preparado para publicação estática na Vercel sem processo de compilação. O domínio principal é `https://ryvo.pt`.

O website institucional e a aplicação são projetos independentes: este repositório publica `ryvo.pt`; a aplicação mantém-se em `app.ryvo.pt`.
