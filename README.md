# RYVO — pacote Vercel sem build

Todos os ficheiros desta pasta devem ficar diretamente na raiz do repositório GitHub.

Na Vercel:

- Framework Preset: `Other`
- Root Directory: deixar vazio
- Build Command: deixar vazio
- Output Directory: deixar vazio

O ficheiro `index.html` tem de aparecer na raiz do repositório, ao lado de `vercel.json`. Não carregues a pasta exterior como uma subpasta do repositório.
